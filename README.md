# Library Management System in C++

A simple console-based Library Management System developed using C++. This project helps manage library records and provides basic operations such as adding books, searching books, issuing books, and returning books.

## Features

- Add new books
- Display all books
- Search books by title
- Issue books
- Return books
- File handling for data storage
- Menu-driven interface
- Object-Oriented Programming using Classes

## Technologies Used

- C++
- File Handling
- Object-Oriented Programming (OOP)

## Learning Outcomes

This project helped in understanding:

- Classes and Objects
- File Handling in C++
- Menu-Driven Programs
- Searching Records
- Updating Records
- Basic Library Management Operations
